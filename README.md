# lab-index-ssl

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或网站，仅作为静态资源索引与存放用途。

## 项目简介

`lab-index-ssl` 是一个实验性质的索引仓库，主要功能是集中管理并发布若干独立的 HTML 文件。所有页面均为自包含的静态内容，不依赖外部服务或数据库。仓库本身不提供任何商业服务或推广内容。

## 目录说明

```
.
├── index.html          # 主索引页面，列出所有归档页面链接
├── pages/              # 存放各独立 HTML 页面
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- `index.html`：仓库的入口页面，提供其他页面的导航。
- `pages/`：各独立 HTML 页面的存放目录，每个文件对应一个独立的归档内容。
- `assets/`：存放页面共用的样式、脚本或图片资源。

## 页面归档说明

- 所有页面均为纯静态 HTML，可直接在浏览器中打开。
- 页面之间相互独立，不共享状态或数据。
- 归档内容可能不定期更新或清理，不保证长期可用。
- 页面内容由维护者自行决定，不代表任何第三方立场。
